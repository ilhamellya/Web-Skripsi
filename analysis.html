Analysis

We collect data from Twitter related to online tutoring applications such as Ruangguru, Pahamify, Quipper, and Cerebrum. The collected data is then processed and analyzed using the Naïve Bayes sentiment classification algorithm to identify positive, negative, and neutral sentiments. The results of this analysis are presented in visualizations for easy understanding, providing an overview of public perception of each application. To view the full conclusions of this analysis, please see the results below.


Data

The collected data consists of tweets containing keywords related to each application, such as 'ruangguru' for the Ruangguru app, 'pahamify' for the Pahamify app, 'cerebrum' for the Cerebrum app, and 'quipper' for the Quipper app. The data includes tweets posted within the last three years. Conclusion

The sentiment analysis of the four online tutoring applications indicates that Quipper has the highest positive sentiment (79.1%), followed by Cerebrum (64.0%), Ruangguru (60.4%), and Pahamify (32.7%). Meanwhile, Pahamify shows the largest proportion of neutral sentiment (38.0%), suggesting more users express no strong opinions. For negative sentiment, Pahamify has the highest percentage (29.3%), followed by Ruangguru (24.9%), Quipper (14.4%), and Cerebrum (12.1%). This highlights the varied public perceptions of these applications.

The performance evaluation of the Naïve Bayes model demonstrates excellent results. Cerebrum achieved the highest model performance, with accuracy, precision, recall, and F1-score all at 95%, followed by Pahamify with 93% across all metrics. Quipper achieved an accuracy of 83%, with slightly lower metrics for the others, while Ruangguru had the lowest performance, with an accuracy of 79% but still showing reasonably high precision (81%). This indicates that the model effectively handled the data, particularly for applications with clearer sentiment patterns.

Overall, the Naïve Bayes classification model successfully categorized sentiments with solid performance. The results reveal that Cerebrum and Quipper tend to have more positive public perceptions compared to the other applications. This research can serve as a reference for prospective users of online learning applications to understand recent public sentiment towards the studied applications, helping them make better-informed decisions.
